1.What is the purpose of module_param() in a kernel module?

Allows you to pass parameters to a kernel module when loading it with insmod or modprobe.




2.What does the third argument (like 0660) control?

Sets file permissions for the parameter in sysfs.

0660 means owner and group can read/write, others cannot.
